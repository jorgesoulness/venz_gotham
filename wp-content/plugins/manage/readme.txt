== Description ==

Manage is a centralized dashboard for monitoring, optimizing, and maintaining multiple WordPress sites without switching between individual site admins.

= CENTRALIZED SITE STATUS VISIBILITY =
Get full visibility into all your sites’ performance and identify security risks at a glance. Manage gives you one place to monitor site health, maintenance needs, and operational status across your portfolio.

= OPTIMIZE PERFORMANCE WITHOUT THE EFFORT =
Keep your sites lean, clutter-free, and performant in a click. Run database optimizations and performance scans across your portfolio without having to log in to each site separately.

= SAVE HOURS ON MAINTENANCE TASKS =
Eliminate site-by-site maintenance overhead by handling updates and optimizations for multiple sites from one dashboard, helping agencies, freelancers, and site owners work more efficiently at scale.

**FEATURES**

* **Performance monitoring**: Track performance scores for speed, SEO, and accessibility to help maintain consistent site quality.
* **Bulk actions**: Update plugins, themes, and WordPress versions across all sites simultaneously.
* **One-click login**: Access any site’s admin instantly without needing to manage or retrieve credentials externally.
* **Security risk detection**: Stay ahead of risks with real-time alerts for plugin security issues.
* **Database cleanup**: Remove transients, comments, post revisions, and auto-drafts to keep sites lean and running fast.
* **Safe updates & rollback**: Update with confidence. If a plugin update causes an issue, quickly revert to the previous version.

= WHY USE MANAGE? =

Manage acts as a professional maintenance checklist, surfacing maintenance issues that are often missed when working only within the standard WordPress admin.

**Efficiency at scale**: Perform bulk updates and optimizations across all sites to eliminate repetitive manual work and reclaim hours of your work week.

**Proactively improve site health**: Improve performance and accessibility directly from the dashboard by leveraging integrated tools like image optimization and accessibility remediation, available only in the premium version.

**Professional workflow**: Maintain a clear history of actions and monitor site health so your sites stay updated, optimized, and secure.

= WHAT’S BEING ADDED NEXT? =

* **Proactive alerting**: Receive real-time notifications for newly detected security risks or critical updates.
* **Uptime monitoring**: Get notified the moment a site goes down so you can resolve issues immediately.
* **Bulk plugin installation**: Effortlessly install and activate new plugins across multiple sites in a single action.
* **Client reporting**: Generate comprehensive maintenance and performance summaries to demonstrate the value of your services to clients.

= Get Started =

For more details, visit our [Help Center](https://elementor.com/help/).

**Get started with a few easy steps**

1. Install the Manage plugin on your WordPress site.
2. Connect the plugin to your Elementor account via Elementor -> Site Management -> Settings.
3. Manage your sites from the “Managed Sites” dashboard at https://my.elementor.com/websites/sites/manage/all/

This plugin requires a connection to an active Elementor account. This connection is initiated manually by the user.

== Frequently Asked Questions ==

= What is Manage? =

Manage is a WordPress plugin with a centralized dashboard that lets you manage multiple WordPress sites from one secure, user-friendly interface. Instead of manually logging into multiple WordPress installations, you can perform bulk updates to WordPress core, themes, and plugins across all your connected sites.

It provides a comprehensive overview of your sites, allowing you to monitor, maintain, and optimize your portfolio from a single dashboard.

= Who is Manage for? =

Manage is built for agencies and freelancers who want structured control over site maintenance and performance. It helps you stay in control of your growing list of clients and multiple websites.

For single-site owners, it provides continuous visibility into performance, accessibility, and security, all from one unified dashboard. Instead of manually checking different tools, you can monitor performance trends, run safe updates, clear database bloat, and keep your site optimized with minimal effort.

Manage centralizes operations for everyone. Run bulk plugin, theme, and WordPress updates, detect security risks, track performance history, and use built-in rollback protection to update WordPress sites with confidence.

= Why should I use Manage instead of handling sites individually? =

Manually switching between WordPress dashboards, updating plugins one by one, and tracking site health separately increases the risk of issues falling between the cracks and wastes valuable time.

Manage lets you monitor multiple websites from one dashboard, keeping everything organized with one comprehensive overview. With bulk updates, one-click rollbacks, and clear performance reports, you can maintain your WordPress websites faster, safer, and more efficiently.

= What kind of performance insights will I get? =

Manage tracks page speed, SEO, and Accessibility using Google Lighthouse, providing specific scores, optimization tips, and a 30-day history of your site’s performance trends.

While the free version includes automated scans every 14 days, the premium version, available through Elementor One, provides automated daily scans and the ability to trigger manual, on-demand scans as often as every 4 hours so you can see the immediate impact of your optimizations.

= Is there a free plan for Manage? =

Yes. The free plan includes unlimited connected sites with view-only performance scans, security risk detection, and one-click login to each site. You can connect as many WordPress websites as you like. There are no site limits.

Premium features such as bulk updates, automatic scans, and database optimization are available with the Elementor One subscription for professional site management.
